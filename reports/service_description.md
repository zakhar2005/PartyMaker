# Текстовое описание веб-сервиса PartyMaker

## 1. Название и назначение
**PartyMaker** — веб-сервис для коллективного выбора фильма в компании друзей.

## 2. Решаемая задача и целевая аудитория
Сервис решает проблему долгого и конфликтного выбора фильма в группе.
Целевая аудитория: студенты, друзья, семьи, небольшие компании.

## 3. Основные функции
1. Создание комнаты хостом и подключение гостей по коду/QR.
2. Раундовое голосование лайк/дизлайк.
3. Подбор фильмов через TMDB API (top rated + discover по жанрам).
4. Определение итогового фильма при единогласном лайке.
5. Генерация QR-кода с информацией о финальном фильме.
6. API генерации QR-кода (`POST /api/qr/generate`).

## 4. Архитектурные особенности
- Backend: Python `http.server` (ThreadingHTTPServer).
- Frontend: HTML/CSS/JavaScript.
- Внешние API:
  - TMDB API для фильмов/жанров;
  - QRServer API для генерации QR-изображений.
- Хранение данных: in-memory структура `ROOMS` в процессе сервера.

## 5. Инструкции по развертыванию
1. Установить Python 3.10+.
2. Задать TMDB ключ:
   - bash: `export TMDB_API_KEY=...`
   - PowerShell: `$env:TMDB_API_KEY="..."`
   - или `.env` с `TMDB_API_KEY=...`
3. Запустить сервер: `python3 server.py`.
4. Открыть:
   - приложение: `http://localhost:4173`
   - Swagger UI: `http://localhost:4173/docs`
