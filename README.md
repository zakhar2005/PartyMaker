# PartyMaker

Прототип веб-сервиса для совместного выбора фильма в комнате.

## Логика подбора (как вы просили)
1. Хост создаёт комнату.
2. Гости присоединяются в ту же комнату.
3. Сервис показывает первые 3 фильма из топа.
4. Каждый участник ставит лайк/дизлайк по каждому из 3 фильмов.
5. По итогам первого раунда вычисляются 3 самых популярных жанра (по разнице лайков и дизлайков).
6. Следующие раунды показывают новые фильмы с приоритетом этих жанров.
7. Раунды повторяются, пока все участники не поставят лайк одному и тому же фильму.
8. Итоговый фильм выводится на главном экране и кодируется в QR.

## Состав проекта
- `index.html` — интерфейс комнат и голосования.
- `styles.css` — стили.
- `app.js` — алгоритм комнат/голосования/подбора/финала.
- `reports/practice_work_1.md` — отчёт.

## Как развернуть локально
```bash
cd PartyMaker
python3 -m http.server 4173 --bind 0.0.0.0
```
Открыть в браузере: `http://localhost:4173`

## Как протестировать вручную
1. Введите имя хоста и нажмите «Создать комнату».
2. Добавьте минимум одного гостя (кнопка «Присоединиться»).
3. Нажмите «Начать голосование».
4. В каждом фильме проставьте всем участникам лайк/дизлайк.
5. Нажмите «Обработать раунд» — система посчитает топ-3 жанра и покажет следующий набор.
6. Повторяйте, пока не появится блок с финальным фильмом и QR.

## Быстрая CLI-проверка
При запущенном сервере:
```bash
curl -I http://localhost:4173/
```
Ожидаемо: `HTTP/1.0 200 OK`.
