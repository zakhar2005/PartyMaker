# PartyMaker

Простой фронтенд-прототип сервиса для совместного выбора фильма в компании.

## Что есть в репозитории
- `index.html` — интерфейс сайта.
- `styles.css` — стили (адаптивная тёмная тема).
- `app.js` — интерактивная логика (комната, рекомендации, QR).
- `reports/practice_work_1.md` — отчёт по практической работе.

## Быстрый запуск локально

### Вариант 1 (рекомендуется): Python HTTP server
Требование: установлен Python 3.

```bash
cd PartyMaker
python3 -m http.server 4173 --bind 0.0.0.0
```

После запуска открой в браузере:
- `http://localhost:4173`

### Вариант 2: VS Code Live Server
1. Открой папку `PartyMaker` в VS Code.
2. Установи расширение **Live Server** (если ещё не стоит).
3. ПКМ по `index.html` → **Open with Live Server**.

## Как протестировать функционал вручную
1. Открой сайт.
2. В блоке «Комната» введи имя и нажми «Создать комнату».
   - Должен появиться код комнаты.
3. В блоке «Предпочтения участников» введи 1–3 любимых фильма и нажми «Получить рекомендации».
   - Должны появиться карточки рекомендаций.
4. В блоке «Финальный выбор + QR»:
   - заполни `Название фильма` и `Ссылка на фильм / трейлер` (обязательные поля),
   - по желанию заполни `Год`,
   - нажми «Сгенерировать QR».
   - Должна появиться QR-картинка.

## Проверка через консоль
Пока сервер запущен:

```bash
curl -I http://localhost:4173/
```

Ожидаемый результат: HTTP-ответ `200 OK`.

## Деплой (статический хостинг)
Этот проект можно развернуть как статический сайт:
- GitHub Pages,
- Netlify,
- Vercel (Static),
- любой Nginx/Apache.

Минимально нужно опубликовать файлы:
- `index.html`
- `styles.css`
- `app.js`
